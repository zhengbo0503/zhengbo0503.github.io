1 Lecture 2 : Examples of Groups

1.1 Revision

Group consist of a non-empty set \(G\) and a binary operator \(*\) on \(G\) such that

  1. (G1) \(\forall g,h,k\in G : g*(h*k) = (g*h)*k\). (Associativity)

  2. (G2) \(\exists e\in G, \forall g\in G: e*g=g*e = g\). (Neutral element)

  3. (G3) \(\forall g\in G, \exists g'\in G : g'*g = g*g' = e\). (Inverse)

Remark 1. Symbols for Group operations:

  • “Multiplicative” symbols: \(*,\times,\cdot,\dots\). Write \(g^{-1}\) instead of \(g'\) for the inverse.

  • “Additive” symbols: \(+,\oplus,\dots\). Write \(-g\) instead of \(g'\) for the inverse.

  • If \((G,*)\) is multiplicative, we often write just \(gh\) instead of \(g*h\).

  • We often write \(G\) instead of \((G,*)\) to denote a group.

1.2 Additive Groups of Numbers

Example 2. \((\mathbf{Q},+), (\mathbf{R},+),(\mathbf{C}, +)\) are groups.

Example 3. Even numbers \(2\mathbf{Z}= \{2n\mid n\in \mathbf{Z}\}\), then \((2\mathbf{Z},+)\) is a group.

Proof. This can be checked by definition

  • Sum of even number is even, hence \(+\) is a binary operator on \(2\mathbf{Z}\).

  • \(+\) is associative (G1); \(0\in 2\mathbf{Z}\) is the neutral element (G2); if \(X\in 2\mathbf{Z}\), then also \(-X \in 2\mathbf{Z}\), and \(X - X = 0\) (G3).

 ◻

Example 4. Odd numbers \(2\mathbf{Z}+ 1 = \{2n+1\mid n \in \mathbf{Z}\}\), \(2\mathbf{Z}+ 1\) with \(+\) is not a group. (\(\star\)) Sum of 2 odd numbers is even, hence \(+\) is not an binary operation on odd numbers.

Example 5. \(+\) is a binary operation on \(\mathbf{N}= \{1,2,\dots\}\), but \((\mathbf{N},+)\) is not a group: no neutral elements, \(0\notin \mathbf{N}\).

1.3 Multiplicative Groups of Numbers

Example 6. Take \(\mathbf{C}^*= \mathbf{C}\setminus \{0\}\) (all non-zero complex numbers), then \((\mathbf{C}^*,\times)\) is a group.

Proof. By definition

  1. \(\times\) is a binary operation on \(\mathbf{C}^*\) (product of non-zero complex number is a non-zero complex number).

  2. \(\times\) is associative. (G1)

  3. \(e = 1 \in \mathbf{C}^*\) is the neutral element, \(1\times z = z\times 1 = z, \forall z\in \mathbf{C}^*\). (G2)

  4. If \(z = \mathbf{C}^*\), then \(z^{-1}= 1/z \in \mathbf{C}^*\). (G3)

 ◻

Example 7. \(\mathbf{Q}^*= \mathbf{Q}\setminus \{0\}\), \(\mathbf{R}^*= \mathbf{R}\setminus \{0\}\), \(\mathbf{R}_+ = \{x\in \mathbf{R}\mid x > 0\}\), then \((\mathbf{Q}^*,\times),(\mathbf{R}^*,\times)\) and \((\mathbf{R}_+,\times)\) are all groups. \((\{-1,1\},\times)\) is a group.

Example 8. \((\mathbf{N},\times)\) is not a group, e.g. \(2\) has no inverse, \(1/2\notin \mathbf{N}\).

1.4 Groups of Residues

\(\mathbf{Z}_n = \{0,1,\dots,n-1\}\). Define addition modulo \(n\): \[\tag{add-mod} a\oplus b = \text{the remainder of $a+b$ divided by $n$}.\] i.e. write \(a + b = q\times n + r\), with \(q\in\mathbf{Z},r\in \mathbf{Z}_n\), then \(r = a\oplus b\).

Lemma 9. Let \(n\in \mathbf{N}\), then \((\mathbf{Z}_n,\oplus)\) is a group. The neutral element is \(0\), and the inverse of \(a\in \mathbf{Z}_n\) is \(n-a\in\mathbf{Z}_n\) if \(a\neq 0\) (the inverse of \(0\) is \(0\)).

Proof. \(\oplus\) is binary operation on \(\mathbf{Z}_n\). \(a+0 = 0+a = a \to a\oplus 0 = 0\oplus a = a\) (G2); (G3) is similar, but (G1) is more complicated. ◻

Remark 10. Multiplicative modulo \(n\): \(a\otimes b\) is the remainder of \(ab\) divided by \(n\). But \((\mathbf{Z}_n,\otimes)\) is not a group, since \(0\) has no inverse. Define \(\mathbf{Z}_n^*= \mathbf{Z}_n\setminus\{0\}\), then \((\mathbf{Z}_n^*,\otimes)\) is a group if and only if \(n\) is prime.

1.5 Groups of Matrices

\(M_n(\mathbf{R})\) is \(n\times n\) matrices with entries in \(\mathbf{R}\). \(\mathop{\mathrm{GL}}(n,\mathbf{R}) = \{A\in M_n(\mathbf{R})|\det(A)\neq 0\}\).

Example 11. \((\mathop{\mathrm{GL}}(n,\mathbf{R}),\times)\) is a group.

Definition 12. \((\mathop{\mathrm{GL}}(n,\mathbf{R}),\times)\) is called the general linear group of degree \(n\) over \(\mathbf{R}\).

Example 13. Similarly, we have \((\mathop{\mathrm{GL}}(n,\mathbf{Q}),\times), (\mathop{\mathrm{GL}}(n,\mathbf{C}),\times)\) are groups. In general, \(\mathop{\mathrm{GL}}(n,K)\) with matrix multiplication is always a group with a field \(K\). E.g. finite field \(\mathbf{Z}_p\) where \(p\) is prime.

2 Lecture 3 : Symmetric Groups

Recall: \(\mathbf{N}_n = \{1,2,\dots,n\}, S_n = \{f:\mathbf{N}_n \to \mathbf{N}_n \mid f \text{ bijective}\}\). The composition is \(f\circ g(x) = f(g(x)), \forall x\in \mathbf{N}_n\).

\((S_n,\circ)\) is a group, with the neutral element \[\label{} \mathrm{id}_n = \begin{pmatrix} 1 & 2 & \cdots & n\\ 1 & 2 & \cdots & n \end{pmatrix}.\]

Definition 14. \((S_n,\circ)\) is called the symmetric group of degree \(n\).

Fact. \(S_n\) has \(n!\) elements.

Example 15. \(n = 1\): \[S_1 = \left\{ \begin{pmatrix} 1 \\ 1\end{pmatrix} \right\}.\] \(n=2\): \[S_2 = \left\{ \begin{pmatrix} 1 & 2 \\ 1 & 2 \end{pmatrix}, \begin{pmatrix} 1 & 2 \\ 2 & 1 \end{pmatrix} \right\}.\] \(n = 3\): \[S_3 = \left\{ \begin{pmatrix} 1&2&3\\1&2&3 \end{pmatrix}, \begin{pmatrix} 1&2&3\\1&3&2 \end{pmatrix}, \begin{pmatrix} 1&2&3\\3&2&1 \end{pmatrix}, \begin{pmatrix} 1&2&3\\2&1&3 \end{pmatrix}, \begin{pmatrix} 1&2&3\\2&3&1 \end{pmatrix}, \begin{pmatrix} 1&2&3\\3&1&2 \end{pmatrix} \right\}\]

Example 16. In \((S_3,\circ)\), let \(f = \begin{pmatrix} 1&2&3\\2&1&3 \end{pmatrix}, g = \begin{pmatrix} 1&2&3\\2&3&1 \end{pmatrix}\), then \[g\circ f = \begin{pmatrix} 1&2&3\\3&2&1 \end{pmatrix},\quad f^{-1} = \begin{pmatrix} 1&2&3\\2&1&3 \end{pmatrix}\].

2.1 Cycle Notation

Definition 17. Let \(\alpha_1,\dots,\alpha_r\in\mathbf{N}_n\) be \(r\) distinct elements. The permutation \(f\in S_n\), defined by \(f(\alpha_1) = \alpha_2, f(\alpha_2) = \alpha_3,\dots, f(\alpha_{r-1}) = \alpha_r, f(\alpha_r) = \alpha_1\), and \(f(\alpha) = \alpha\) for all \(\alpha\notin \{\alpha_1,\dots,\alpha_r\}\), is called a cycle of length \(r\), or an \(r\)-cycle. We write \(f= (\alpha_1\ \alpha_2\ \cdots\ \alpha_r)\). A \(2\)-cycle \((\alpha_1\ \alpha_2)\) is called a transposition.

Example 18.

  1. A \(1\)-cycle, \((\alpha)\), is the identity, \(\mathrm{id}_n\).

  2. In \((S_3,\circ)\): \[(1\ 3) = \begin{pmatrix} 1 & 2&3\\ 3&2&1 \end{pmatrix},\quad (1\ 3\ 2) = \begin{pmatrix} 1 & 2&3 \\ 3& 1& 2 \end{pmatrix}\]

Definition 19. Two cycles \((\alpha_1\ \alpha_2\ \cdots\ \alpha_r)\) and \((\beta_1\ \beta_2\ \cdots\ \beta_s)\) are disjoint if \[\{\alpha_1,\dots,\alpha_r\}\cup \{\beta_1,\dots,\beta_s\} = \emptyset.\]

Example 20. \((1\ 2\ 4)\) and \((3\ 5)\) in \(S_5\) are disjoint. \((1\ 2\ 3)\) and \((3\ 4\ 5)\) are not disjoint.

Lemma 21. Let \((\alpha_1\ \alpha_2\ \cdots\ \alpha_r)\) and \((\beta_1\ \beta_2\ \cdots\ \beta_s)\) be two disjoint cycles, then \[(\alpha_1\ \alpha_2\ \cdots\ \alpha_r)\circ(\beta_1\ \beta_2\ \cdots\ \beta_s) = (\beta_1\ \beta_2\ \cdots\ \beta_s)\circ(\alpha_1\ \alpha_2\ \cdots\ \alpha_r).\] i.e. Disjoint cycles commutes.

Proof. Write \(f=(\alpha_1\ \cdots\ \alpha_r)\) and \(g = (\beta_1\ \cdots\ \beta_s)\) , we must show that \(f\circ g = g\circ f\), i.e. \(\forall x\in \mathbf{N}_n,\ f(g(x)) = g(f(x))\). We have three cases

  1. \(x\in \{\alpha_1,\dots,\alpha_r\}\), then also \(f(x) = \{\alpha_1,\dots,\alpha_r\}\). Thus, \(x,f(x)\notin \{\beta_1,\dots,\beta_s\}\), we get \(g(x)=x\), \(g(f(x)) = f(x)\). Hence \(f(g(x)) = f(x) = g(f(x))\).

  2. \(x\in \{\beta_1,\dots,\beta_s\}\), omitted.

  3. \(x\in \mathbf{N}_n\setminus \left\{ \alpha\mid \alpha\in \{\alpha_1,\dots,\alpha_r\}\cup \{\beta_1,\dots,\beta_s\} \right\}\), then \(f(x) = x = g(x)\), hence \(f(g(x)) = f(x) = x = g(x) = f(g(x))\).

 ◻

Theorem 22. Every permutation in \(S_n\) can be written as a product of disjoint cycles.

Remark 23.

  1. “Product” refers to the operation of \(S_n\), i.e. “\(\circ\)”. One also says that \(f\) is a “composite” of disjoint cycles.

  2. Elements of \(\mathbf{N}_n\) that are fixed by \(f\) can be included as \(1\)-cycles, or not. e.g. we can write \((1\ 3)(2)\) or \((1\ 3)\).

  3. The product of this disjoint cycle for \(f\) is unique up to inclusion of \(1\)-cycles and order of the cycles.

  4. When all \(1\)-cycles are included, the length of the cycles add up to \(n\).

Example 24. \(f = \begin{pmatrix} 1&2&3&4&5&6 \\ 3&6&2&5&4&1 \end{pmatrix}\). Then \(f = (1\ 3\ 2\ 6)\circ(4\ 5) = (4\ 5)\circ(1\ 3\ 2\ 6).\)
